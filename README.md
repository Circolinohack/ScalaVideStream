# Scala Video Stream
Video Stream Test. 

## Tech Stack
- Scala
- SBT
- Akka HTTP

### Prerequisites
- [Java](https://www.java.com/it/download/)
- [SBT](https://www.scala-sbt.org/download.html)

### How to Run It
From the command line, go to the root of the project and type:
```
sbt run
```
then go to http://127.0.0.1:8080/stream

Enjoy!
